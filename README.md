# FishMagazineCoverGenerator (釣果画像から釣り雑誌の表紙風画像を作成する)

釣果（ブツ持ち）写真の「魚」をAIで自動検出し、ブラウザ上で釣り雑誌の表紙風画像を作成するWebアプリケーションです。サーバーコストを極限までゼロに抑えるために、完全なクライアントサイドで動作します。

## 特徴
- **AIによる自動推論**: `Transformers.js`（`clipseg-rd64-refined`）を使用し、AIの画像セグメンテーション処理をブラウザ上のWeb Workerで実行します。ユーザーがアップロードした画像が外部サーバーに送信されることは一切ありません。
- **Canvas APIを活用した画像合成加工**: 自動で切り抜かれた画像のアルファ処理（透過）や、背後の穴埋め（簡易インペインティング）、重心を中心とした拡大・縮小処理などをCanvas上でスムーズに行います。
- **快適な操作性**: Vite + React、Tailwind CSSを用いて、モダンで直感的なUI（スライダー調整やドラッグ＆ドロップ対応）を構築しています。

## 使い方

1. リポジトリをクローンまたはダウンロードし、ディレクトリに移動します。
2. Node.js環境（Node.js 18などで動作確認済み）で依存パッケージをインストールします。
   ```bash
   npm install
   ```
3. ローカル開発サーバーを起動します。
   ```bash
   npm run dev
   ```
4. ブラウザでターミナルに表示されたURL（例: `http://localhost:5173/`）にアクセスし、お手持ちの「釣果写真」をアップロードしてください。

> **注意**: 初回のAI検出実行時、バックグラウンドで数十MB程度のモデルファイルのダウンロードが行われます。2回目以降は即時に処理されます。
